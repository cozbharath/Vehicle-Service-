## Installation

1. Clone the repository:
```bash
git clone https://github.com/cozbharath/Vehicle-Service-.git
cd Vehicle-Service-
```

2. Build the project:
```bash
# Using Maven (if applicable)
mvn clean install

# Or using Gradle (if applicable)
gradle build
```
